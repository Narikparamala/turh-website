# **App Name**: Tirupati Universe Electronics

## Core Features:

- Service Showcase: Display the range of electronic repair services offered (AC, TV, Fridge, Washing Machine, etc.).
- Location and Contact Information: Display the shop's location on a map, phone numbers with click-to-call functionality, WhatsApp chat link, and operating hours.
- Training Program Details: Provide information about the apprenticeship program for teenagers, including curriculum details, duration, fees, and safety & consent information.
- Booking Form: Allow customers to submit repair requests through a booking form with fields for personal details, service requirements, and preferred time slots.
- Training Enrollment Form: Enable prospective trainees to enroll in the apprenticeship program through a dedicated form, capturing student information, parental consent, and program preferences.
- Gallery of Repairs and Workshop: Showcase a visual gallery of completed repair projects, workshop images, and YouTube video embeds.
- AI-Powered Problem Diagnostics Tool: Implement a tool using a generative AI model that helps users troubleshoot their electronic devices by providing potential solutions based on a description of the device's problem.

## Style Guidelines:

- Primary color: Navy (#0B3D91) to convey trustworthiness and professionalism.
- Background color: Cream (#F7F5EF) to create a warm and inviting feel.
- Accent color: Red (#E84B3A) for call-to-action buttons and important highlights.
- Body and headline font: 'Inter' sans-serif for a modern and readable appearance.
- Use clean and simple icons to represent different services and website sections.
- Implement a mobile-first, responsive layout that adapts seamlessly to different screen sizes.
- Use subtle animations for loading screens, transitions, and button hovers to enhance user engagement.
